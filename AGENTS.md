## Project Summary
A high-converting, modern landing page for Orchids Global, a shipping and logistics company. The design focuses on a clean corporate aesthetic with deep blue and cyan accents, featuring interactive elements and a prominent shipment tracking focal point.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Architecture
- `src/app/`: Main page and layout
- `src/components/`: Modular UI components (Hero, Features, Stats, HowItWorks, Trust, Footer)
- `public/`: Static assets (images, fonts)

## User Preferences
- Clean Corporate look with plenty of white space.
- Rounded corners (large) and subtle drop shadows.
- Deep Professional Blue (#0F172A) and Cyan accents.
- Reveal-on-scroll animations using Framer Motion.

## Project Guidelines
- Follow Next.js App Router conventions.
- Keep components modular and reusable.
- Ensure full responsiveness across all device sizes.
- No comments unless requested.

## Common Patterns
- Scroll-triggered animations using `whileInView`.
- Gradient overlays for professional image backgrounds.
- High-impact typography with bold headings and tracking-tighter classes.
